Use Case UC01 – Login

- Tên Use Case: Login
- Mục tiêu: Cho phép user truy cập hệ thống
- Tác nhân: Admin/ Employee
- Điều kiện trước: Có tài khoản hợp lệ
- Luồng chính (Happy Flow):
    1. Mở trang Login
    2. Nhập username/password
    3. Click Login
    4. Chuyển vào Dashboard
- Luồng phụ (Alternative Flow): Sai password → Hiển thị “Invalid credentials”
- Điều kiện sau: Dashboard hiển thị
