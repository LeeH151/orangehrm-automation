# Nghiên cứu về Playwright và pytest

## Playwright
- Locator: Cách Playwright tìm kiếm phần tử trên UI.
- POM (Page Object Model): Cấu trúc code giúp tổ chức các hành động UI.

## pytest
- Fixtures: Tạo môi trường test.
- Markers và Parametrize: Quản lý nhóm test và dữ liệu cho test.
- Hooks: Tùy chỉnh quá trình chạy test.
